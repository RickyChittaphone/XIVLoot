Welcome to XIVLoot.com!

XIVLoot.com is a website that makes easier the management of gear distribution during raid progression. This website was made by raiders who wanted to steer away from the old habit of using spreadsheets while not making the experience worst. XIVLoot offers similar functionalities to the other existing methods of gear tracking but also offers more! Read the "About" sections for more details

The website is still in development, so if you encounter any bug or if you have any recommendations feel free to contact using your prefered method below :

How to use XIVLoot.com

Create a static with any name above. This will create a static with 8 players.

The recommended first step is for you and everyone else in your static to each claim a player, as otherwise anyone with the link of the static can modify any unclaimed player.

You can then edit your player's name, job and gear.

You can then modify the static's settings such as : PGS settings, Lock settings, etc.

If you require any help please drop by the discord server and we will be happy to assist!
